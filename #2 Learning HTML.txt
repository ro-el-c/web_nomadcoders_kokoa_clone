2) Head

- 모든 html 문서는 <!DOCTYPE html>로 시작
- <html> </html> 사이의 모든 텍스트가 html 코드
  - lang : 웹사이트의 주된 언어 (검색 엔진에 도움 주기 위함)
- 웹사이트 구조 -> 1. head 2. body


1. head : 웹사이트의 환경 설정
- 외부로 보여지지 않음
- 페이지에 관한 설정 - ex. title = 페이지 이름(상단)

- <meta> : 부가 정보
  - title★, base, link, style, script => 여러 종류 데이터 표현
  - name / content
    -> name="description" ★ : 구글이 검색할 때 찾는 태그
  - property = "og:image" : 카카오톡 등 sns에 링크 공유시 보여지는 이미지 (썸네일 느낌)
    -> fb : 페이스북 / twitter : 트위터 / nate : 네이트온
- <charset> ★ : 한글 등 문자 표시 가능
  - utf-8 항상 넣을 것

위 작업들) 사이트의 정보들을 최대한 명확하게 브라우저에게 알려주는 용도
- 어떤 언어, 사이트의 description, title 등이 무엇인지


2. body : 사용자가 볼 수 있는 content
- 위에서 지금까지 배웠던 모든 태그/생성자가 내용이 됨
