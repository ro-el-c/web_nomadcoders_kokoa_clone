### tag&attribute (태그와 속성)
1. 제목
- h1 ~ h6

2. 리스트
- li : list item _ 리스트에 포함되는
- ol : ordered list _ 순서 X
- ul : unordered list _ 순서 O

3. 링크
- a : anchor(닻) _ = link _ 다른 웹사이트로 이동
- href : HTTP/hyperlink reference _ 이동할 곳
- target : 어느 탭에 열릴건지
  -> default="_self" / 다른 탭="_blank"

4. 이미지 
: <img>닫는 태그가 없음 = sel-closing tag
- img
- src : img의 컨텐츠 _ 이미지 주소 또는 위치
